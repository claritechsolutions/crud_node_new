1.  get API : To get all data-
	get: http://localhost:3000/api/courses


2. To insert Data:
	post : http://localhost:3000/api/courses/req
	
	request body ==>
	var req = {
	name: 'xyz',
	email:'abc@gmail.com',
	password:'agagya',
	cPassword:'aagaya'	
	}

	
3. To fetch particular record:
	get : http://localhost:3000/api/courses/3
	

4. Delete Record:
	delete : http://localhost:3000/api/courses/3
